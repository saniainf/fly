   Name: TiledMax TMX Parser for .Net 4.0
Version: 1.0.1
--------------------------------------------------------------

Welcome to the first edition of TiledMax, a .Net 4.0 parser fo the Tiled
map format (.tmx).

In this download you'll find an example, and the binary libraries to use
this parser.

CHANGE LOG ---------------------------------------------------

v1.01 - Added support for zlib, making the most recent
        maps readable.
